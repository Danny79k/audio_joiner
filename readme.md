per chi lo deve usare, semplicemente importare i file audio dentro del folder e cambiare i nomi dentro FILES[] per rispecchiare i file audio che si vogliono mischiare

## Installazione

serve python installato

- git clone https://github.com/Danny79k/audio_joiner

- cd audio_joiner

- python -m venv venv\Scripts\activate

- python -m pip install -r requirements.txt

## eseguire il mixup

python main.py
